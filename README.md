## 🚀 Featured Projects

### [edge-ai-performance-benchmark](https://github.com/Aanalpatel99/edge-ai-performance-benchmark)
Real-time Edge AI benchmarking system — multithreaded PySide6 + pyqtgraph desktop 
dashboard measuring inference latency and FPS under varying hardware constraints. 
Dedicated worker thread with frame-dropping mechanism, structured CSV logging, 
and automated comparative analysis.  
`Python` `PySide6` `pyqtgraph` `OpenCV` `MediaPipe` `Multithreading`

### [uav-safety-companion](https://github.com/Aanalpatel99/uav-safety-companion)
Autonomous UAV safety state machine (SAFE / WARNING / FAILSAFE) built on 
PX4 SITL + Gazebo + ROS2. Real-time sensor fault detection, automated state 
transitions, and telemetry monitoring dashboard. *(Active development)*  
`C++` `Python` `ROS2` `PX4 SITL` `Gazebo` `Ubuntu`

### [embedded-ota-firmware-system](https://github.com/Aanalpatel99/embedded-ota-firmware-system)
Production-grade A/B partition OTA firmware update system on embedded Linux. 
SHA-256 hash verification, RSA signature validation, watchdog acknowledgment 
within 30 seconds of reboot, and automatic rollback to stable partition on failure.  
`C` `Bash` `Embedded Linux` `OTA` `Cryptography` `Firmware`

### [tinyml-facial-recognition-esp32](https://github.com/Aanalpatel99/tinyml-facial-recognition-esp32)
Custom 4-layer depthwise separable CNN deployed on ESP32-CAM with 520KB total SRAM. 
Grayscale 96×96 input, INT8 quantization (1.2MB → 95KB weights in flash), 
TFLite Micro tensor arena optimization — final RAM usage ~105KB.  
`Embedded C` `TFLite Micro` `TinyML` `ESP32` `INT8 Quantization` `CNN`
